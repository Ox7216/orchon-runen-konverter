Schreib- und Leseregeln

Hier sind wichtige Regeln zum Schreiben und Lesen von Orchon Runen

  • 1. Harte Vokale werden für harte Silben, leichte Vokale für leichte Silben
    
𐰉𐰀𐰉𐰀 = Baba → harte Silben → (a) wird verwendet
𐰋𐰀𐰋𐰀 = Bebe → weiche Silben → (e) wird verwendet
  • 2. Wenn zwei Konsonanten direkt nebeneinanderstehen, werden sie nicht als eine Silbe, sondern mit ihren Namen gelesen.
    
𐰺𐰲 = araç
𐰲𐰺 = açar
  • 3. Wenn ein Vokal und ein Konsonant nebeneinanderstehen, bilden sie eine Silbe.
    
𐰾𐰇 = sü
𐰇𐰾 = üs 
Nicht: esü oder ües
  • 4. Die Wörter werden beim Silbentrennen nach harten und weichen Lauten getrennt
    
𐰴𐰀𐰠𐰀 → 𐰴𐰀 + 𐰠𐰀 = Kale
𐰾𐰀𐰺𐰀 → 𐰾𐰀 + 𐰺𐰀 = Sera
                                
Mehrdeutige Wörter:
𐰖𐰺𐰣 → kann verschieden gelesen werden: Ayran, Yaran, Yarın
                                
Zur Klarstellung wird ein zusätzlicher Vokal eingefügt:
𐰀𐰖𐰺𐰀𐰣 = Ayran
𐰖𐰀𐰺𐰣 = Yaran
𐰖𐰀𐰺𐰀𐰣 = Yarın
  • 5. Wörter werden nicht nach Silben, sondern nach Wortstämmen und Endungen (Morphemen) getrennt.
    
Yanak (Wange):
Lateinisch: ya + nak (nach Vokalen)
Morphemisch: yan (Stamm: Seite) + ak (Suffix)

𐰖𐰀𐰣𐰴 = Yanak
𐰖𐰀𐰣𐰶 = Yanık
